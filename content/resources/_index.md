A curated collection of resources for learning about and working with semantic technologies.

## Tutorials & Training

- **[OBO Academy](https://oboacademy.github.io/obook/)** - Comprehensive training materials for ontology development, curated by the OBO community
- **[SSSOM Tutorial](https://mapping-commons.github.io/sssom/)** - Learn how to create and share semantic mappings using the SSSOM standard
- **[LinkML Documentation](https://linkml.io/)** - Modern framework for authoring schemas and data models

## Tools We Maintain

- **[Ontology Development Kit (ODK)](https://github.com/INCATools/ontology-development-kit)** - Docker-based toolkit for ontology development workflows
- **[sssom-py](https://github.com/mapping-commons/sssom-py)** - Python library for working with SSSOM mappings
- **[ROBOT](http://robot.obolibrary.org/)** - Command-line tool for ontology operations (contributor)
